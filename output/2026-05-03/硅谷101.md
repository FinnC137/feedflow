# 硅谷101 — 2026-05-03

【DeepSeek V4 三项核心技术：CSA+HCA、mHC 与 Muon 优化器】

DeepSeek V4 在工程的完成度上带来非常大的惊喜。
核心创新有三：CSA（压缩稀疏注意力）加 HCA（重度压缩注意力），
在注意力机制上大幅降低计算开销；
mHC（流形约束超连接）优化模型内部信息流动；
Muon 优化器则进一步提升了训练效率。
三者叠加使得 DeepSeek V4 在 token efficiency 上继续一骑绝尘，
计算成本压缩到竞品的 1/3、内存占用降到 1/10。
V4 支持 100 万 token 上下文，面向 agentic coding 和复杂多步任务，
被外界视为中国 AI 在模型架构层面的重要突破。

【中美技术路径分歧：资源约束倒逼效率创新】

硅谷的主流路径是更多 GPU、更大模型、更强闭源产品。
而中国在芯片受限的资源约束下，被倒逼走出一条效率优先的路线。
这种差异正从被动变为主动——DeepSeek 证明了效率本身可以成为竞争力。
当 token efficiency 被提升到系统级高度，
模型架构、存储系统、runtime 调度、通信与深度定制都成为优化空间。
嘉宾指出，异构计算将是未来常态：
训练和推理对芯片的需求正在分化，不再"一卡打天下"。

【Token Efficiency 是 AGI 的必经之路】

嘉宾肖志斌明确提出：token efficiency 是达到 AGI 或更强 agent system 的基础条件。
没有效率，AGI 只能是个 demo；有了效率，AGI 才能成为真正的产品和基础设施。
随着 AI 步入 Agentic 时代，模型需要处理复杂多步任务，
每次推理的 token 消耗呈数量级增长。
如果不能将单 token 成本降到足够低，
agent 的规模化部署在商业上就无法成立。
这一判断也被 Jenny Xiao 从资本市场角度呼应：
未来的赢家不只取决于谁的模型最聪明，
而取决于谁能用最低成本、最稳定地把智能交付给最多开发者和企业。

【DeepSeek 为美国闭源模型划定"死亡线"】

前 OpenAI 研究员、Leonis Capital 合伙人 Jenny Xiao 给出了一个尖锐判断：
DeepSeek 带来的最大风险，是为美国基础模型公司划定了一条"死亡地带"。
如果你是一家基础模型公司，却被开源公司超越了，你的业务价值基本上就是零。
开源模型代表的是更高效率、更低成本、更开放生态的路径，
这是美国闭源商业模式面临的最大结构性威胁之一。
她强调"效率也是智能的一部分"——这个认知正在被硅谷主流接受，
但能否在产品层面做出回应仍是未知数。

【Anthropic 崛起与 OpenAI 的路线分歧】

Jenny Xiao 指出 Anthropic 正强势崛起，
核心驱动力是 Claude Code 在编程场景的深度渗透和企业级信任的积累。
"赢编程者赢 AGI"成为一个值得关注的命题——
几乎所有计算机任务都可以被形式化为编程，
而编程任务的可衡量性使得模型能力可以被清晰评估和优化。
与之对比，OpenAI 走大而全路线，GPT-5.5 定价贵出一倍，
在市场对 ROI 日益敏感的背景下是否还能站得住脚存在疑问。
公开市场对 Anthropic 的胃口比对 OpenAI 大得多，
而不少机构在试图于 IPO 前抛售 OpenAI 股份。

【开源蚕食闭源收入，混合部署成为主流】

开源模型的快速进步正在实质性地侵蚀闭源模型的收入基础。
嘉宾观察到混合部署正成为企业的主流选择：
核心高价值任务用最强闭源模型，常规任务转向开源方案。
xAI 和 Meta 各面临困境：马斯克频繁变阵导致战略不稳定，
Meta 虽然坚持开源但缺乏差异化壁垒。
DeepSeek 被报道已适配华为昇腾等国产芯片，
如果生态成型，将为中国 AI 产业链提供从芯片到模型到应用的完整闭环。

【大模型 IPO 前夜的隐忧：投资者更在乎 ROI】

在资本市场层面，大模型公司的 IPO 正面临关键考验。
投资者从追逐技术叙事转向追问实际回报，
ROI 取代模型 benchmark 成为估值核心。
DeepSeek 选择在此时融资并深度绑定华为昇腾，
被认为是在重塑中国 AI 生态的战略布局——
从单点模型能力延伸到芯片适配、软件栈和开发者生态的全链条竞争。
这标志着一个范式的转变：
大模型竞争正在从单点 benchmark 变成系统战争，
模型架构、token efficiency、芯片适配、商业化、开源生态
正在变成同一场战争的不同战场。
